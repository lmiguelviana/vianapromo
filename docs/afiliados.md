# Guia de Programas de Afiliados — Viana Promo

## Resumo: funcionam no WhatsApp?

| Plataforma | Link direto no WhatsApp | Rastreamento | Cookie | Comissão | Precisa developer? |
|---|---|---|---|---|---|
| **Mercado Livre** | ✅ Sim | ✅ Funciona | 30 dias | 2–12% | ❌ Não |
| **Amazon Brasil** | ✅ Sim | ✅ Funciona | 24h (carrinho: 90d) | 2–10% | ❌ Não |
| **Shopee** | ✅ Sim | ✅ Funciona | 7 dias | 2–10%+ | ❌ Não |
| **Magazine Luiza** | ✅ Sim | ✅ Funciona | 30 dias | 2–8% | ❌ Não |
| **TikTok Shop** | ⚠️ Parcial | ❓ Incerto | Variável | 5–20%+ | ❌ Não |

---

## Mercado Livre

- Cadastro: **afiliados.mercadolivre.com.br** — aprovação imediata, sem developer
- Gerar link: painel próprio + extensão de navegador
- Link funciona colado direto no WhatsApp ✅
- WhatsApp é canal aceito nos termos ✅
- Não tem API para automação de links (precisa gerar manualmente no painel)

## Amazon Brasil

- Cadastro: **associados.amazon.com.br** — aprovação condicional (precisa de 3 vendas em 180 dias)
- Gerar link: SiteStripe (barra no topo da Amazon) ou PA API 5.0
- Link funciona direto no WhatsApp ✅
- **PA API 5.0**: a mais completa — busca produtos e já gera link com seu tag. Requer aprovação separada após as 3 vendas.
- Cookie de carrinho de 90 dias protege a comissão mesmo com cookie de sessão de 24h

## Shopee

- Cadastro: **shopee.com.br/m/shopee-afiliado** ou **involve.asia**
- Gerar link: direto pelo app Shopee (botão "Compartilhar" > "Link de afiliado") — mais simples de todos
- Link funciona direto no WhatsApp ✅
- WhatsApp explicitamente permitido nos termos ✅
- Involve Asia oferece API para feeds de produtos

## Magazine Luiza (Parceiro Magalu)

- Cadastro: **parceiro.magalu.com.br** — aprovação imediata
- Modelo diferente: você recebe uma mini-loja virtual com sua URL personalizada
- Link da loja funciona no WhatsApp ✅
- WhatsApp incentivado pela própria Magalu ✅
- Menos prático para enviar produto a produto (é uma loja, não link direto)

## TikTok Shop

- Cadastro via TikTok (pode exigir mínimo de seguidores)
- Links projetados para uso **dentro do TikTok**, não no WhatsApp
- Rastreamento de comissão **não garantido** fora do app TikTok ⚠️
- **Evitar como canal principal via WhatsApp**

---

## Sobre o disclosure obrigatório (CONAR/CDC)

Sempre indique que o link é de afiliado na mensagem. Exemplo:
> 🔗 *link de afiliado — se comprar por aqui me ajuda sem custo extra pra você!*

Isso é legalmente exigido no Brasil e gera mais confiança nos grupos.

---

## Recomendação de ordem para começar

1. **Shopee** — link gerado em segundos pelo app, WhatsApp permitido, comissão boa
2. **Mercado Livre** — maior catálogo do Brasil, link simples, 30 dias de cookie
3. **Amazon** — cookie de carrinho de 90 dias é o maior diferencial, PA API permite automação futura
4. **Magazine Luiza** — boa para categorias de eletro/casa onde a Magalu é forte
5. **TikTok Shop** — usar só se tiver conta TikTok ativa com seguidores
